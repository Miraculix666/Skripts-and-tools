# 🛠️ Skripts-and-tools

> **Standard**: v2.0.0 (Master Agent)
> **Description**: Consolidated collection of administrative scripts and tools for DevOps and IT infrastructure.

## 📂 Project Structure

- `docs/` - Project documentation, analysis reports, and changelog.
- `scripts/` - Categorized scripts:
    - `active-directory/` - AD user and rights management.
    - `analysis/` - System and file analysis tools.
    - `network/` - WOL, PXE, and topology reports.
    - `system/` - Configuration, recovery, and optimization.
- `OPSI/` - OPSI package management (sub-project).
- `Launcher/` - Custom script launchers.

## 🚀 Usage

Navigate to the respective subfolder in `scripts/` to find the tool you need. All scripts follow the Antigravity v2.0.0 code standards.
