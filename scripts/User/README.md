# User & Remote Session Management Scripts

This directory contains session launchers, multi-RDP utilities, and classroom dashboard launchers.

## Scripts

- **Manage-RDPUsers.ps1**: Manages access rights and remote desktop permissions for local client environments.
- **Manage-RDPUsers.md**: Documentation and command reference for the RDP user manager.
- **Start-MultiRDP.ps1**: Generates and launches connection profiles for MultiRDP dashboards.
- **Start-RemoteSessions.ps1**: Establishes remote management sessions for administrative tasks.
- **quick_rdp.html**: Interactive HTML dashboard for launching remote sessions.
