# 🏗️ ADVisualizer

> **Standard**: v2.0.0 (Master Agent)
> **Description**: Modular PowerShell project for Active Directory user analysis and visualization.

## 📂 Project Structure

- `docs/` - Project documentation, sources, and changelog.
- `Modules/` - Core logic in PowerShell modules (`.psm1`).
- `scripts/` - Entry point scripts for report generation.
- `src/` - Supporting source code (if any).

## 🚀 Getting Started

The main entry point for generating user-group reports is located in `scripts/Export-ADUserGroups.ps1`.

For more detailed information, please refer to:
- [README.md](docs/README.md) - Detailed usage and setup.
- [CHANGELOG.md](docs/CHANGELOG.md) - Version history.
